# SABLE-HE Research v0.7.0

Phase 6 adds stronger formal proof and security-reduction materials.

## Highlights

- New `sable.proofs` module.
- CLI commands: `proof-info`, `security-game`, `proof-obligations`, `proof-ledger`, and `proof-package`.
- Formal IND-CPA game definition with public evaluation keys.
- Proof obligations for correctness and security.
- Hybrid proof ledger for evaluation-key and challenge-ciphertext replacement.
- Deployment-aware sample-count ledger.

## Scope

This release strengthens proof presentation and review readiness. It does not certify concrete parameters or replace independent cryptanalysis.
