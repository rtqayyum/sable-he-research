# SABLE-HE Research v0.6.0 — Phase 5 Standardization and External Review

This release adds public tooling for external cryptanalysis and pre-standardization discussion.

## Added

- `sable.standardization` module.
- CLI commands:
  - `standardization-info`
  - `standardization-readiness`
  - `assumptions-spec`
  - `parameter-template`
  - `review-checklist`
  - `review-package`
- Phase 5 documentation under `docs/phase5/`.
- Review-package generator for public JSON/Markdown/CSV review artifacts.
- Phase 5 tests and example script.

## Scope

This is **not** a certification release. It does not claim NIST, ISO, HE.org, FIPS, CAVP, or CMVP validation. It prepares SABLE-HE for external cryptanalysis and pre-standardization discussion.
