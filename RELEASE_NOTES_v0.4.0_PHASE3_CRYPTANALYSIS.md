# SABLE-HE Research v0.4.0 — Phase 3 Independent Cryptanalysis Release

This release adds a clean public independent-cryptanalysis package:

- `sable.cryptanalysis` module.
- `sable-he cryptanalysis-info` CLI.
- `sable-he cryptanalysis-bundle` CLI.
- Public surface accounting for expansion keys, compaction keys, row-difference surfaces, and encrypted traffic.
- Reproducible JSON/Markdown review bundles.
- Phase 3 documentation and tests.

This release does not include private LaTeX papers, generated PDFs, raw validation logs, or internal research drafts.
