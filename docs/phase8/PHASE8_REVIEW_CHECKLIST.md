# Phase 8 reviewer checklist

Reviewers should check:

1. Are the exposed sample counts correct for each public key and deployment model?
2. Are sparse q-ary LPN rows modeled with the right support distribution?
3. Does any optimization create additional plaintext-canceling relations?
4. Are low-noise settings rejected or escalated for review?
5. Are ISD estimates compared against modern Prange/Stern/Dumer/BJMM-style literature?
6. Are q-ary BKW estimates adapted correctly to the field size and sample regime?
7. Are quantum estimates stated only as conservative proxies?
8. Are candidate parameter rows clearly marked as review targets, not certified parameters?
