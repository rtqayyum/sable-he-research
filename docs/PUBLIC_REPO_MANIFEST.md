# Public repository manifest

Included:

- `src/sable/` package source;
- `tests/` public test suite;
- `examples/` clean examples;
- `docs/` user and review documentation;
- `.github/` workflows and public issue templates;
- packaging metadata.

Excluded:

- `paper/`, `papers/`, `latex/`, `*_latex/`;
- all PDFs and LaTeX build artifacts;
- rendered pages and private manuscript drafts;
- generated logs and large experiment outputs;
- old wheel/source-distribution files;
- secrets and credentials.
