# External Review Package

Use:

```bash
sable-he review-package --output sable_phase5_review_package
```

The generated package contains:

- machine-readable standardization manifest;
- readiness matrix;
- security-assumption specification;
- parameter-set templates;
- attack-surface reports;
- reviewer checklist;
- cryptanalysis disclosure template.

Send the generated package together with the exact Git tag, PyPI version, and the public paper/specification to independent reviewers.
