# Threat Model

The prototype models passive chosen-plaintext security for a research candidate under sparse-LPN and q-ary-LPN assumptions. It does not provide:

- chosen-ciphertext security;
- malicious evaluator verification;
- side-channel resistance;
- secure memory handling;
- production-grade randomness management;
- certified parameter sets.

The code is suitable for algebraic validation and estimator experiments only.
