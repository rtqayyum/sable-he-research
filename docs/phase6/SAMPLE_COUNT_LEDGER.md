# Sample-count ledger

The sample ledger records public surfaces that must be included in any security estimate:

- expansion-key sparse-LPN rows;
- expansion sparse-entry upper bound;
- compaction-key q-ary LPN rows;
- same-entry row-difference samples;
- deployment-dependent input ciphertext rows.

The ledger is not an attack estimator. It is the accounting substrate that parameter estimators and external reviewers should use.
