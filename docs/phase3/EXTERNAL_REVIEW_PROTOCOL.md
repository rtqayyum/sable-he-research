# External Review Protocol

1. Checkout the exact Git tag.
2. Install the exact PyPI wheel or build from source.
3. Generate the cryptanalysis bundle.
4. Reproduce all JSON/Markdown reports.
5. Attack the surfaces using independent tooling.
6. Report attacks using the disclosure template or GitHub cryptanalysis issue template.

Acceptance before production security claims requires at least two independent reports, stable parameters, reproducible test vectors, hardened implementation review, and side-channel review.
