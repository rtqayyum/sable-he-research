# Attack Surface Specification

SABLE-HE exposes public noisy-linear-equation surfaces through the expansion key, compaction key, and encrypted traffic. The key review question is whether these surfaces remain hard under sparse q-ary LPN and q-ary LPN/code assumptions for the selected sample counts.

The bundled estimator is a rejection screen. It is intentionally conservative and should be replaced or supplemented by specialized cryptanalysis before any security claim.
