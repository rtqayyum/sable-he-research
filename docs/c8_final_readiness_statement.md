# C8 final readiness statement

SABLE-HE-C7/C8 is ready as:

- a research artifact;
- an internal validation prototype;
- an independent-cryptanalysis package;
- a pre-standardization candidate package.

SABLE-HE-C7/C8 is not yet ready as:

- production cryptography;
- a certified secure parameter set;
- a FIPS/NIST-approved algorithm;
- a standardization-approved implementation.

The reason is not lack of code volume. The reason is that cryptographic trust is
built from independent attack analysis, parameter validation, implementation
hardening, and third-party evaluation. Those cannot be replaced by a designer's
claim.

The next external milestone should be independent cryptanalysis of the C7
coordinate compactor and the sparse-LPN/q-ary-LPN parameter search.
