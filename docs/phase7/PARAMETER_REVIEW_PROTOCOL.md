# Parameter review protocol

External parameter review should check:

1. Sparse q-ary LPN hardness for the expansion-key surface.
2. Dense q-ary LPN/code-decoding hardness for the compaction surface.
3. Row-difference and relation-derived public sample surfaces.
4. Low-noise clean-subset attacks.
5. Prange/Stern/Dumer/May-Ozerov-style ISD estimates.
6. BKW and q-ary/coded-BKW variants.
7. Impact of FL deployment sample counts.
8. Whether seeded/streamed compaction can reduce storage without changing the attack surface.

A reviewer should not treat the included internal screens as final security estimates.
