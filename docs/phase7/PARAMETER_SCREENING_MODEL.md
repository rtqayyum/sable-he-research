# Parameter Screening Model

The Phase 7 screening model reports:

1. fresh expanded row support;
2. depth row-support growth;
3. conservative GSW-style evaluation error bound;
4. compaction q-ary piling-up bound;
5. one-replica and replicated failure bounds;
6. public expansion-key sparse-LPN rows;
7. public compaction-key q-ary LPN/code rows;
8. same-entry compaction row-difference rows;
9. no-clean-subset and Prange-style finite screens.

These diagnostics are intended to reject obviously weak candidate rows and to help external cryptanalysts reproduce the public sample surfaces. They do not model all known or future LPN/ISD/BKW attacks.
