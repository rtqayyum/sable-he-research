# SABLE-HE v0.9.0: Phase 8 strengthened attack-estimator framework

This internal release adds reproducible LPN/ISD/BKW screening tools for candidate parameter sets.

New commands:

- `sable-he attack-estimator-info`
- `sable-he attack-surfaces`
- `sable-he attack-grid`
- `sable-he attack-estimate`
- `sable-he attack-package`

The estimator supports manuscript strengthening and external review preparation. It does not certify parameters.
